Texte Explicatif

JavaScript est un langage de programmation utilisé pour rendre les pages web interactives. PHP est un langage côté serveur utilisé pour générer du contenu dynamique sur le web.

JavaScript est exécuté dans le navigateur de l'utilisateur, tandis que PHP est exécuté sur le serveur avant que la page ne soit envoyée au navigateur.

Exemples

Exemple de script JavaScript interactif


<!-- HTML -->
<button id="myButton">Cliquez-moi</button>
<p id="output"></p>

<!-- JavaScript -->
document.getElementById('myButton').addEventListener('click', function() {
    document.getElementById('output').innerText = 'Bonjour, monde!';
});

Exemple de script PHP côté serveur


<!-- PHP -->
<?php
echo "Bonjour, monde!";
?>
