Texte Explicatif

CSS (Cascading Style Sheets) est un langage utilisé pour décrire la présentation des documents HTML. Il permet de styliser les éléments HTML avec des couleurs, des polices, des espacements, etc.

Les sélecteurs CSS permettent de cibler des éléments HTML spécifiques pour les styliser. Par exemple, le sélecteur de balise p cible tous les paragraphes.

Exemples

Exemple de styles CSS appliqués

Ce paragraphe est stylisé avec CSS.


/* Sélecteur de balise */
p {
    color: blue;
    font-size: 16px;
}

/* Sélecteur de classe */
.styled-element {
    border: 1px solid #333;
    padding: 10px;
    background-color: #f0f0f0;
}
