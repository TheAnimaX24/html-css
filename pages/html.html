Texte Explicatif

HTML (HyperText Markup Language) est le langage standard utilisé pour créer des pages web. Il utilise des balises pour structurer le contenu d'une page.

Les balises HTML sont entourées de chevrons (<>). Par exemple, une balise de paragraphe est <p>.

Exemples

Exemple de page HTML basique


<!DOCTYPE html>
<html>
<head>
    <title>Ma première page</title>
</head>
<body>
    <h1>Bienvenue sur ma page!</h1>
    <p>Ceci est un paragraphe.</p>
</body>
</html>
